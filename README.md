## HNAV

    Usage: hnav [-atwci] URL [OPT=VAL ...]
    
    This program helps launching the browser.
    
    -a : Open in application mode.
    -t : Open in tabbed mode.
    -w : Open in windowed mode.
    -c : Open in cookie gathering mode.
    -i : Open in incognito mode.
    
    -f : Enforce Icecat/Firefox.
    -m : Enforce Mozilla Firefox, no other one.
    -g : Enforce Google chrome.
    -s : Enforce Surf.
